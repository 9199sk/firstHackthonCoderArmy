Introduction

HTML provides several ways to organize and display content. Two of the most important structures are lists and tables.

Nested Lists

Lists can be nested inside other lists to create hierarchical structures.

Example 1: Basic Nested List

  • Main item 1
    • Sub item 1.1
    • Sub item 1.2
  • Main item 2
    • Sub item 2.1
    • Sub item 2.2

Example 2: Mixed List Types

  1. First topic
    • Subtopic A
    • Subtopic B
  2. Second topic
    • Subtopic X
    • Subtopic Y

Example 3: Complex Nesting

  • Category A
    1. Item A1
      • Detail A1a
      • Detail A1b
    2. Item A2
  • Category B
    1. Item B1
    2. Item B2

Example 4: Multi-level Nesting

  • Level 1
    • Level 2
      • Level 3
        • Level 4

Introduction to HTML Tables

Tables are used to display data in a grid format with rows and columns.

Rows and Columns

Tables consist of rows (horizontal) and columns (vertical). Each intersection is called a cell.

Table Example 1: Basic Table

Row 1, Cell 1 Row 1, Cell 2
Row 2, Cell 1 Row 2, Cell 2

Table Tags

The <table> tag defines a table. Inside it, we use:

  • <tr> for table rows
  • <td> for standard cells
  • <th> for header cells

Adding Borders to a Table
