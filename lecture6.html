Media Tags

HTML provides special tags for embedding audio and video content directly into web pages.

The <audio> and <video> tags make it easy to include media without needing external plugins.

Integrating Audio Files

Use the <audio> tag to embed audio content:

<audio src="audiofile.mp3">
  Your browser does not support the audio element.
</audio>

Adding Controls to Audio

Add the controls attribute to show playback controls:

<audio src="audiofile.mp3" controls>
  Your browser does not support the audio element.
</audio>

Audio Example with Controls

Audio: Proper Formats

Different browsers support different audio formats. Provide multiple sources for compatibility:

  • MP3 - Widely supported
  • OGG - Open format
  • WAV - Uncompressed, large file size

Why Proper Formats are Needed

Different browsers support different audio formats. By providing multiple sources, you ensure your audio will play across all browsers.

<audio controls>
  <source src="audiofile.mp3" type="audio/mpeg">
  <source src="audiofile.ogg" type="audio/ogg">
  Your browser does not support the audio element.
</audio>

Integrating Video Files

Use the <video> tag to embed video content:

<video src="videofile.mp4" width="320" height="240" controls>
  Your browser does not support the video tag.
</video>

Video Example with Controls

Video: Proper Formats

Like audio, different browsers support different video formats:

  • MP4 - Most widely supported
  • WebM - Open format, good quality
  • OGG - Less common for video

Allow Fullscreen in Videos

Add the allowfullscreen attribute to allow fullscreen viewing:

<video src="videofile.mp4" controls allowfullscreen>
  Your browser does not support the video tag.
</video>

Autoplay Feature

Use the autoplay attribute to play media automatically (use sparingly):

<video src="videofile.mp4" autoplay muted>
  Your browser does not support the video tag.
</video>

Note: Most browsers require the muted attribute for autoplay to work.

Embedding YouTube Videos

You can embed YouTube videos using the <iframe> tag:

<iframe width="560" height="315" 
src="https://www.youtube.com/embed/VIDEO_ID" 
frameborder="0" allowfullscreen>
</iframe>

Multi-Page Websites

Most websites consist of multiple HTML pages linked together.

Creating Multi-Page Websites

Create separate HTML files for each page and link them together using the <a> tag:

<a href="index.html">Home</a>
<a href="about.html">About</a>
<a href="contact.html">Contact</a>

Header Section

The header typically contains the site logo and navigation menu:

<header>
  <h1>My Website</h1>
  <nav>
    <a href="index.html">Home</a>
    <a href="about.html">About</a>
    <a href="contact.html">Contact</a>
  </nav>
</header>

Main Section

The main content area contains the unique content for each page:

<main>
  <h2>Welcome to Our Website</h2>
  <p>This is the home page content.</p>
</main>

Footer Section

The footer typically contains copyright information and additional links:

<footer>
  <p>&copy; 2023 My Website. All rights reserved.</p>
</footer>

Creating Copyright Symbol

Use the HTML entity &copy; to create the copyright symbol: ©

Deploying a Website on Netlify

Netlify is a popular platform for deploying static websites.

Steps to Deploy on Netlify:

  1. Create a Netlify account at netlify.com
  2. Drag and drop your website folder onto the Netlify deployment area
  3. Or connect your GitHub repository for automatic deployments
  4. Netlify will provide you with a unique URL for your website
  5. You can customize the domain name in the site settings

Benefits of Using Netlify:

  • Free for basic usage
  • Automatic deployments from Git
  • SSL certificates included
  • Global CDN for fast loading
  • Form handling and serverless functions
