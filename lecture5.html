What is an HTML Form?

HTML forms are used to collect user input and send it to a server for processing.

Forms contain various input elements like text fields, checkboxes, radio buttons, and more.

Form Structure

form Tag

The <form> tag defines an HTML form that contains input elements.

<form action="/submit" method="post">
  <!-- Form elements go here -->
</form>

input Tag

The <input> tag creates various form controls depending on its type attribute.

<input type="text" name="username">

label Tag

The <label> tag defines a label for an input element, improving accessibility.

<label for="username">Username:</label>
<input type="text" id="username" name="username">

Name & Value Attributes

The name attribute identifies the data when submitted to the server.

The value attribute specifies the initial value for an input field.

<input type="text" name="firstname" value="John">

Radio Buttons

Radio buttons allow users to select exactly one option from a group.

<input type="radio" id="male" name="gender" value="male">
<label for="male">Male</label>

<input type="radio" id="female" name="gender" value="female">
<label for="female">Female</label>

for Attribute

The for attribute of a label should match the id attribute of the input it describes.

Checkboxes

Checkboxes allow users to select zero or more options from a set.

<input type="checkbox" id="subscribe" name="subscribe" value="yes">
<label for="subscribe">Subscribe to newsletter</label>

Other Input Types

Placeholder Attribute

Provides a hint about the expected value of an input field.

<input type="text" placeholder="Enter your name">

Date Input

<label for="birthday">Birthday:</label>
<input type="date" id="birthday" name="birthday">

File Input

<label for="myfile">Select a file:</label>
<input type="file" id="myfile" name="myfile">

Email Input

<label for="email">Email:</label>
<input type="email" id="email" name="email">

Telephone Input

<label for="phone">Phone:</label>
<input type="tel" id="phone" name="phone">

Textarea Tag

The <textarea> tag creates a multi-line text input control.

<label for="message">Message:</label>
<textarea id="message" name="message" rows="4" cols="50">
Default text goes here
</textarea>

Select and Option Tags

The <select> tag creates a drop-down list.

The <option> tag defines an option in the drop-down list.

<label for="cars">Choose a car:</label>
<select id="cars" name="cars">
  <option value="volvo">Volvo</option>
  <option value="saab">Saab</option>
  <option value="mercedes">Mercedes</option>
  <option value="audi">Audi</option>
</select>

Button Tag

The <button> tag creates a clickable button.

<button type="submit">Submit Form</button>
<button type="reset">Reset Form</button>
<button type="button">Regular Button</button>

Required Attribute

The required attribute specifies that an input field must be filled out.

<input type="text" name="username" required>

Wrapping Up

HTML forms are essential for collecting user input on websites.
