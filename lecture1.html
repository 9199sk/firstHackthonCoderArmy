What is the Internet?

The internet is a global network of interconnected computers that communicate using standardized protocols.

It allows for the sharing of information and resources across vast distances.

How is Data Transferred?

Data is transferred over the internet using packet switching technology.

Information is broken down into small packets, sent individually across the network, and reassembled at the destination.

This process uses protocols like TCP/IP to ensure reliable delivery.

What is an IP Address?

An IP (Internet Protocol) address is a unique numerical identifier assigned to each device connected to a network.

It serves two main functions:

  1. Network interface identification
  2. Location addressing

Example IP address: 192.168.1.1

How to Access a Website

When you type a website address (URL) into your browser:

  1. Your browser contacts a DNS server to translate the domain name to an IP address
  2. A connection is established with the server at that IP address
  3. The server sends the requested web page back to your browser
  4. Your browser renders the web page for you to view

Who Assigns the IP Address?

IP addresses are assigned by:

  • ICANN (Internet Corporation for Assigned Names and Numbers) - globally
  • ISPs (Internet Service Providers) - to end users
  • Network administrators - within private networks

Concept of DNS

DNS (Domain Name System) is often called the "phonebook of the internet."

It translates human-friendly domain names (like www.example.com) into IP addresses that computers use to identify each other.

In-depth Working of DNS

The DNS resolution process involves multiple steps:

  1. Check browser cache
  2. Check operating system cache
  3. Check router cache
  4. Check ISP's DNS servers
  5. Recursive search through DNS hierarchy if needed

This process happens in milliseconds to quickly resolve domain names to IP addresses.

Problems in Data Transfer and MAC Address

During data transfer, several issues can occur:

  • Packet loss
  • Network congestion
  • Data corruption
  • Routing errors

MAC Address

A MAC (Media Access Control) address is a unique identifier assigned to a network interface controller.

Unlike IP addresses, MAC addresses are permanently assigned to hardware during manufacturing.

MAC Address Format

MAC addresses are 48-bit addresses typically represented as six groups of two hexadecimal digits.

Example: 00:1A:2B:3C:4D:5E

The first three octets represent the manufacturer identifier, while the last three are device-specific.

What are Port Numbers?

Port numbers are 16-bit numbers used to identify specific processes or services on a device.

They allow a single device to host multiple network services simultaneously.

Common Port Numbers

  • Port 80: HTTP (Web traffic)
  • Port 443: HTTPS (Secure web traffic)
  • Port 25: SMTP (Email)
  • Port 22: SSH (Secure shell)

Port Number Format

Port numbers range from 0 to 65535, divided into three categories:

  1. Well-known ports (0-1023)
  2. Registered ports (1024-49151)
  3. Dynamic/private ports (49152-65535)

IPv4 & IPv6

IPv4 (Internet Protocol version 4) is the fourth version of the Internet Protocol.

It uses 32-bit addresses, allowing for about 4.3 billion unique addresses.

Example: 192.168.1.1

IPv6

IPv6 (Internet Protocol version 6) is the most recent version, designed to replace IPv4.

It uses 128-bit addresses, allowing for a virtually unlimited number of unique addresses.

Example: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

Local Area Network, Switch & Router

A LAN (Local Area Network) is a network that connects devices in a limited geographical area.

Network Devices

Switch: Connects devices within a LAN and forwards data to the specific device.

Router: Connects different networks together and routes data between them.

Public and Private Addresses

Public IP addresses are routable on the internet and unique globally.

Private IP addresses are used within private networks and are not routable on the internet.

Common private IP ranges:

  • 10.0.0.0 - 10.255.255.255
  • 172.16.0.0 - 172.31.255.255
  • 192.168.0.0 - 192.168.255.255

Modern Routers

Modern routers often combine multiple functions:

  • Routing between networks
  • Network address translation (NAT)
  • Firewall protection
  • Wireless access point
  • Switch functionality

What is a VPN?

A VPN (Virtual Private Network) extends a private network across a public network.

It enables users to send and receive data across shared or public networks as if their devices were directly connected to the private network.

Benefits of VPNs:

  • Enhanced security and privacy
  • Bypassing geographic restrictions
  • Secure remote access to network resources

What is Web Development?

Web development is the work involved in developing websites for the internet or an intranet.

It can range from developing simple static web pages to complex web applications.

Front-end vs. Back-end

Front-end development involves creating what users see and interact with in their browsers.

Back-end development involves server-side programming and database management.

What is the MERN Stack?

The MERN stack is a popular JavaScript-based technology stack for building web applications.

MERN stands for:

  • MongoDB - A NoSQL database
  • Express.js - A web application framework for Node.js
  • React - A JavaScript library for building user interfaces
  • Node.js - A JavaScript runtime environment

Who is a Full-Stack Developer?

A full-stack developer is proficient in both front-end and back-end development.

They can work with databases, servers, systems engineering, and client-side user interfaces.

Wrapping Up

Understanding internet fundamentals is crucial for web development.

From how data travels across networks to how websites are built, these concepts form the foundation of modern web technologies.

Whether you're focusing on front-end, back-end, or full-stack development, this knowledge will help you build better, more efficient web applications.
